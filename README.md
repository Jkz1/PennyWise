## ğŸ” Firebase Configuration

This project uses Firebase for authentication and database services.

To run the app locally, you must configure your own Firebase project:

- Create a project in the Firebase Console

- Enable Authentication and Cloud Firestore

- Add Firebase configuration files to the project:

  - android/app/google-services.json

  - ios/Runner/GoogleService-Info.plist

These files are excluded from version control for security reasons
